## 2.PostgreSQL

使用env配置文件

端口: `51003:5432`

```shell
docker run -d --name postgresql-service \
  --network DockerCustomNet \
  --ip 173.1.10.2 \
  --user "$(id -u)":"$(id -g)" \
  --privileged=true \
  -e TZ=Asia/Shanghai \
  -p 51003:5432 \
  -v /data/container/volume/postgresql-service/data:/var/lib/postgresql/data \
  --env-file /data/container/env.d/postgres.env \
  --restart always \
  postgres:16.1
```
